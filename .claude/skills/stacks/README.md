# Stack skills

These are optional, framework-specific conventions.
Delete any you don't use — they add to your context window cost.

Available: prisma-patterns, express-api, react-frontend, symfony-api, ci-cd-pipeline
Contributions welcome for: FastAPI, Django, Rails, Go, Rust, Laravel, Phoenix

Note: `ci-cd-pipeline` lives here because its content is tool-specific
(GitHub Actions / GitLab CI YAML, action pinning, Node/Python examples).
Downstream users on self-hosted Jenkins, Drone, Woodpecker, CircleCI, or
anything else should delete this skill — the concepts transfer but the
snippets will mislead Claude into generating the wrong YAML shape.
